Spotify Music Analytics Dashboard
Project Overview

Spotify Music Analytics Dashboard is a real-time interactive data analytics project developed using Python, Streamlit, Plotly, Pandas, and Spotify API.

The project allows users to search for artists and analyze song-related insights such as popularity trends, release timelines, artist distribution, and music analytics through interactive visualizations.

This dashboard demonstrates practical implementation of:
- API Integration
- Data Cleaning
- Real-Time Data Fetching
- Data Visualization
- Dashboard Development
- CSV Export Functionality
- Interactive UI Development

The project was built to simulate real-world data analyst workflows involving data extraction, transformation, visualization, and dashboard deployment.

# ⚙ Installation & Setup

 Clone Repository

```bash
git clone https://github.com/Sarthak05G/Spotify_Analytics.git
```

---

Navigate to Project Folder

```bash
cd Spotify_Analytics
```

---

 Install Dependencies

```bash
pip install -r requirements.txt
```

---

 Run Streamlit App

```bash
streamlit run app.py
```

---
 Spotify API Setup

1. Create Spotify Developer Account  
2. Generate:
   - Client ID
   - Client Secret
3. Replace credentials inside:

```python
client_id = "YOUR_CLIENT_ID"
client_secret = "YOUR_CLIENT_SECRET"
```
